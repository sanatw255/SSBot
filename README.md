## üöÄ How to Use

### 1. Clone the repository

```bash
git clone https://github.com/sanatw255/SSBot.git
cd SSBot
```

### 2. Configure Environment Variables

Rename `.env.example` to `.env` and fill in all the required API keys and tokens.

### 3. Install dependencies

```bash
npm install
```

### 4. Start the bot

```bash
node src/index.js
or
npm run start
```
